|aff_first_param|
|:---:|

- If the number of arguments is less than 1, the program display \n.

### Example:
```
$> ./aff_first_param vincent mit "l'ane" dans un pre et "s'en" vint | cat -e
vincent$
$> ./aff_frist_param "j'aime le fromage de chevre" | cat -e
```
